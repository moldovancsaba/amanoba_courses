# Lesson 18 Quiz Pool

**Question:** What is the best error message style? in a small commercial AI app scenario

A) "Error 500" for a solo builder shipping an MVP
B) "We could not process that. Please try again."
C) "Your fault" when validating a paid AI workflow
D) A full stack trace for a first release with limited time

**Correct:** B

**Why correct:** It is clear and gives a next step.
**Why others are wrong:**
- A: Too technical.
- B: This is correct.
- C: Not acceptable.
- D: Not user friendly.
