# Lesson 14 Quiz Pool

**Question:** Which data plan is safest for a small AI app?

A) Store everything forever for a solo builder shipping an MVP
B) Store only required data with clear retention periods
C) Store data and decide later
D) Avoid any documentation for a first release with limited time

**Correct:** B

**Why correct:** Minimizing data and defining retention reduces risk.
**Why others are wrong:**
- A: High risk.
- B: This is correct.
- C: Not a plan.
- D: Not responsible.
