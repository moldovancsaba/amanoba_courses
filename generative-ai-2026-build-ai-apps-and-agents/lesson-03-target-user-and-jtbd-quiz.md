# Lesson 3 Quiz Pool

**Question:** Which JTBD statement is most outcome focused and testable?

A) "Use AI to improve workflows for teams"
B) "A freelance designer turns rough client notes into a proposal draft in under 10 minutes"
C) "Build an AI platform that does many tasks"
D) "Help people write better emails"

**Correct:** B

**Why correct:** It names a user, input, output, and time bound outcome.
**Why others are wrong:**
- A: Too vague.
- B: This is correct.
- C: Feature focused.
- D: Outcome is vague.

**Tags:** #jtbd #difficulty-easy #type-scenario

**Question:** A persona reads, "Any remote worker who uses AI." What is the best fix?

A) Add more demographics like age and gender
B) Add role, workflow context, and constraints
C) Add more tools and integrations
D) Add a longer bio story for a first release with limited time

**Correct:** B

**Why correct:** Role and context make the persona actionable.
**Why others are wrong:**
- A: Demographics alone do not define workflow.
- B: This is correct.
- C: Tools are not the persona.
- D: Length does not improve clarity.
