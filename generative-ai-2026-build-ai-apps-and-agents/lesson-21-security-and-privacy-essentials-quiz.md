**Question:** What is the best place to store API keys?

A) In client side code for a solo builder shipping an MVP
B) In env variables on the server
C) In a public gist when validating a paid AI workflow
D) In a README for a first release with limited time

**Correct:** B

**Why correct:** Env variables keep secrets out of code.
**Why others are wrong:**
- A: Exposes secrets.
