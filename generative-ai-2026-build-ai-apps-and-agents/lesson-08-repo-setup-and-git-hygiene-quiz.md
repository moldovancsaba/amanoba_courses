# Lesson 8 Quiz Pool

**Question:** Which README first line is most useful? in a small commercial AI app scenario

A) "Welcome to my project" for a solo builder shipping an MVP
B) "An app that turns call notes into a proposal draft for freelance designers"
C) "This repository is cool"
D) "AI app" for a first release with limited time

**Correct:** B

**Why correct:** It names the user and outcome clearly.
**Why others are wrong:**
- A: Too generic.
- B: This is correct.
- C: Not informative.
- D: Too vague.
