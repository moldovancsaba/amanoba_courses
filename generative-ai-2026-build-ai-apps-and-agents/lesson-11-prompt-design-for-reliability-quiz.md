# Lesson 11 Quiz Pool

**Question:** Which prompt spec is most reliable? in a small commercial AI app scenario

A) "Write something about proposals"
B) "You are a proposal assistant. Create a 6 bullet summary in this format: Title, Client, Goal, Constraints, Steps, Next action"
C) "Use AI to help with work"
D) "Make it better" for a first release with limited time

**Correct:** B

**Why correct:** It provides role, task, and clear output format.
**Why others are wrong:**
- A: Vague task.
- B: This is correct.
- C: Too vague.
- D: No context or format.
