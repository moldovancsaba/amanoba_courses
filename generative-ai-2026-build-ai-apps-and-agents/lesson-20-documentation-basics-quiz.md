**Question:** Why keep docs short for early users? in a small commercial AI app scenario

A) Short docs are more likely to be read and used
B) Long docs look more professional
C) Docs do not matter when validating a paid AI workflow
D) Short docs replace support

**Correct:** A

**Why correct:** Short docs are easier to consume and act on.
**Why others are wrong:**
- A: This is correct.
- B: Not true.
- C: Not true.
- D: Docs reduce but do not replace support.
