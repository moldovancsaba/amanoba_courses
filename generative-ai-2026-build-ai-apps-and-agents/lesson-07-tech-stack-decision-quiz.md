# Lesson 7 Quiz Pool

**Question:** Which stack choice best supports a fast MVP for a solo builder?

A) Multiple microservices with custom Kubernetes
B) A simple web app with hosted deployment and Stripe
C) A full data lake and BI tooling
D) A multi cloud architecture with custom auth

**Correct:** B

**Why correct:** It minimizes setup and supports payments.
**Why others are wrong:**
- A: Too complex.
- B: This is correct.
- C: Not needed for MVP.
- D: Too heavy.
