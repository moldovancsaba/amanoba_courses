# Lesson 5 Quiz Pool

**Question:** A team wants a first paid release. Which feature set best fits MVP scope?

A) A full dashboard, analytics suite, and AI assistant
B) The single workflow that delivers the promised outcome
C) Every integration users might ask for
D) A complete marketplace for partners

**Correct:** B

**Why correct:** MVP should focus on the one outcome users pay for.
**Why others are wrong:**
- A: Too large for a first release.
- B: This is correct.
- C: Not required for the promise.
- D: Not MVP.

**Tags:** #mvp #difficulty-easy #type-scenario

**Question:** Which rule best decides must have features?

B) It maps directly to the product promise outcome

**Why others are wrong:**
- A: Not outcome based.
