# Lesson 17 Quiz Pool

**Question:** Why use scenarios to test an agent? in a small commercial AI app scenario

A) They replace user feedback
B) They mimic real usage and reveal failures
C) They increase token usage
D) They remove the need for prompts

**Correct:** B

**Why correct:** Scenarios reveal how the agent behaves in real conditions.
**Why others are wrong:**
- A: Not a replacement.
- B: This is correct.
- C: Not a benefit.
- D: Not true.
