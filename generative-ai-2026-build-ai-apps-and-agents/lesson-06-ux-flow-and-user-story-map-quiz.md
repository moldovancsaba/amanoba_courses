# Lesson 6 Quiz Pool

**Question:** Which start to finish flow best matches a proposal draft app?

A) Login, upload notes, generate draft, review, export
B) Login, browse blog, view analytics, export
C) Login, set theme, invite team, open settings
D) Login, read FAQ, view roadmap, logout

**Correct:** A

**Why correct:** It matches the core workflow and outcome.
**Why others are wrong:**
- A: This is correct.
- B: Not related to outcome.
- C: Focuses on settings, not outcome.
- D: Not an outcome flow.

**Tags:** #user-flow #difficulty-easy #type-scenario

**Question:** A flow includes 12 steps. What is the best correction?

A) Add two more for completeness
B) Remove steps that do not change user state
C) Keep all steps to show features
D) Replace the flow with a feature list

**Correct:** B

**Why correct:** Flows should focus on state changes tied to outcomes.
**Why others are wrong:**
- A: Adds complexity.
- B: This is correct.
- C: Not outcome based.
- D: Does not describe user journey.
