**Question:** Ni njia gani salama ya kushughulikia API keys kwenye Vercel?

A) Kuzihardcode kwenye code.
B) Kuzihifadhi kwenye env variables.
C) Kuzituma kwa client kwa urahisi.
D) Kuzihifadhi kwenye README.

**Correct:** B

**Why correct:** Env variables hulinda secrets bila kuziweka kwenye code.
**Why others are wrong:**
- A: Hardcode huongeza hatari.
- C: Client si salama.
- D: README inaweza kuvuja.
