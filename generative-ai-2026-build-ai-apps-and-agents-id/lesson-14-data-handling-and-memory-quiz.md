**Question:** Apa praktik terbaik untuk penanganan data pengguna?

A) Mengumpulkan semua data tanpa tujuan.
B) Menentukan tujuan dan retensi untuk setiap data.
C) Menyimpan data selamanya.
D) Tidak menulis kebijakan data.

**Correct:** B

**Why correct:** Tujuan dan retensi memberi kontrol dan transparansi.
**Why others are wrong:**
- A: Risiko tinggi dan tidak perlu.
- C: Tidak aman dan tidak perlu.
- D: Kebijakan tetap diperlukan.
