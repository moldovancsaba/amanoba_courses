**Question:** Mengapa empty state penting di UI AI dalam konteks app AI komersial.?

A) Untuk menyembunyikan input.
B) Untuk membimbing pengguna sebelum memasukkan data.
C) Untuk menambah langkah tanpa manfaat.
D) Untuk mengganti helper text.

**Correct:** B

**Why correct:** Empty state memberi konteks awal.
**Why others are wrong:**
- A: Tidak sesuai fungsi.
- C: Tidak perlu.
- D: Helper text tetap dibutuhkan.
