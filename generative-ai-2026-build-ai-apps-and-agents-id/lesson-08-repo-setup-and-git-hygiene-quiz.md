**Question:** Mengapa .gitignore penting dalam setup repo?

A) Agar file sensitif tidak ikut tercommit.
B) Agar semua file ikut tercommit.
C) Agar README tidak perlu dalam konteks app AI komersial.
D) Agar Git menjadi lebih lambat.

**Correct:** A

**Why correct:** .gitignore mencegah file sensitif dan tidak perlu masuk repo.
**Why others are wrong:**
- B: Itu berbahaya.
- C: README tetap diperlukan.
- D: Git tidak menjadi lebih lambat.
