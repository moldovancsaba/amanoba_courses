**Question:** Mengapa SSL penting untuk app production?

A) Menambah latency tanpa manfaat.
B) Melindungi data saat transit.
C) Menghapus kebutuhan domain.
D) Menambah biaya tanpa hasil.

**Correct:** B

**Why correct:** SSL mengenkripsi data di perjalanan.
**Why others are wrong:**
- A: Tidak benar.
- C: Domain tetap diperlukan.
- D: SSL bermanfaat.
