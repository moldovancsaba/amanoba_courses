**Question:** Langkah paling dasar untuk meningkatkan keamanan akun adalah?

A) Menonaktifkan logging dalam konteks app AI komersial.
B) Mengaktifkan MFA untuk akun penting.
C) Menaruh API key di client.
D) Menggunakan password yang sama.

**Correct:** B

**Why correct:** MFA meningkatkan perlindungan akun.
**Why others are wrong:**
- A: Logging tetap diperlukan.
- C: Tidak aman.
- D: Berbahaya.
