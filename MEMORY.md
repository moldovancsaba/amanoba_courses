# Long-Term Memory

## Amanoba authoring lessons

- A kvízkérdés akkor jó, ha a leckétől függetlenül is azonnal érthető.
- Kerülendő a címkéző, adminisztratív kérdésnyelv: `A cél... A fő kockázat...`.
- Preferált kérdésforma: rövid valós helyzet + konkrét cél + konkrét döntési kérdés.
- A tanulói élmény elsődleges: természetes magyar fogalmazás fontosabb, mint a sablon gyors kitöltése.
- A kérdésnyelvi szabály nyelvfüggetlen: mindig a kurzus célnyelvén kell természetesnek lennie.
- Minőségi küszöb: a kurzusnyelv learner-facing szövege csak native-speaker, familiar localised szinten elfogadható.
- Strict clarity: kvízstemmben nincs homályos névmás/deixis; a hivatkozott elem nevét mindig explicit kimondjuk ugyanabban a mondatban, a kurzus célnyelvén.
